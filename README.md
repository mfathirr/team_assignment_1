# Sistem Playlist Lagu

Sistem aplikasi playlist lagu berbasis OOP (Object-Oriented Programming) yang dibangun menggunakan Java. Aplikasi ini memungkinkan pengguna untuk mengelola playlist lagu dengan dua tipe pengguna: Admin dan Member.

## Fitur Utama

- **Login Admin**: Admin dapat menambahkan lagu-lagu baru ke dalam playlist
- **Login Member**: Member dapat melihat daftar playlist dan mencari lagu yang tersedia
- **Manajemen Playlist**: Menambah dan menampilkan daftar lagu dalam playlist
- **Pencarian Lagu**: Fitur pencarian untuk menemukan lagu berdasarkan judul

## Cara Menjalankan

### Prasyarat
- Java Development Kit (JDK) versi 8 atau lebih tinggi
- Command Prompt atau Terminal

### Langkah-Langkah

1. **Clone repository**:
   ```bash
   git clone <repository-url>
   cd team_assignment_1
   ```

2. **Pastikan Java sudah terinstal**:
   ```bash
   java -version
   ```

3. **Jalankan program**:
   ```bash
   java PlaylistOOP.java
   ```

4. **Ikuti menu yang ditampilkan** di aplikasi

## Struktur File

- `PlaylistOOP.java` - File utama program (Main class)
- `User.java` - Class dasar untuk pengguna
- `Admin.java` - Class untuk pengguna Admin
- `Member.java` - Class untuk pengguna Member
- `Lagu.java` - Class untuk data lagu
